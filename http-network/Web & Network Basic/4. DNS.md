# DNS (Domain Name System)

- 사람이 외우기 힘든 IP 주소를 사람에게 친숙한 문자열로 변환
- 도메인 명에서 IP 주소로, IP 주소에서 도메인 명으로 조사하는 서비스를 제공한다.
